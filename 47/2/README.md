### Traditionell funktion
````javascript 
function add100(a) {
  return a + 100
}
````

### Arrow funktion (tre olika varianter)

Variant 1
````javascript
const add100 = (a) => {
  return a + 100
}
````

Variant 2
````javascript
const add100 = (a) => a + 100
````

Variant 3
````javascript
const add100 = a => a + 100
````
